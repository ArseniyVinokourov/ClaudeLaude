# Code of Conduct

## Short version

Be respectful. Don't be a jerk. Assume good intent. If someone's behavior makes contributing unpleasant, the maintainer will step in.

## Scope

This applies to issues, pull requests, and any project communication channel.

## Enforcement

The maintainer ([@ArseniyVinokourov](https://github.com/ArseniyVinokourov)) has final say. Violations — harassment, trolling, sustained disruption — result in warnings, then bans. Decisions are not negotiable.

## Reporting

Email [arseny@tentixo.com](mailto:arseny@tentixo.com). Reports are confidential.
